# P1.3 - Import/Export CSV/XLSX ✅

## Résumé de l'implémentation

Le système d'import/export permet de gérer en masse les dossiers et fichiers d'archives via des fichiers Excel (.xlsx) ou CSV (.csv).

## 📦 Composants créés

### 1. Dépendances installées

- **phpoffice/phpspreadsheet v5.2.0** : Bibliothèque pour manipuler fichiers Excel et CSV
  - Installation avec `--ignore-platform-req` pour contourner ext-gd et ext-zip manquantes

### 2. Services (src/Service/)

#### ExportService.php (330 lignes)

**Fonctionnalités** :

- `exportDossiers($dossiers, $format)` : Export dossiers vers Excel/CSV
- `exportFichiers($fichiers, $format)` : Export fichiers vers Excel/CSV
- `generateDossiersTemplate()` : Template pré-formaté pour import dossiers
- `generateFichiersTemplate()` : Template pré-formaté pour import fichiers

**Caractéristiques** :

- Style professionnel (en-têtes colorés, polices grasses)
- Auto-dimensionnement des colonnes
- Séparateur CSV : `;` / Enclosure : `"`
- Génération dans répertoire temporaire système

#### ImportService.php (360 lignes)

**Fonctionnalités** :

- `importDossiers($filepath)` : Import dossiers depuis Excel/CSV
- `importFichiers($filepath)` : Import fichiers depuis Excel/CSV

**Validation** :

- Champs obligatoires vérifiés (libellé, département_id/dossier_id, type)
- Vérification existence entités référencées (département, dossier parent, règle rétention, boîte physique)
- Format dates validé (YYYY-MM-DD)
- Gestion erreurs/warnings avec numéro de ligne

**Retour** :

```php
[
    'success' => 15,      // Nombre d'entités importées
    'errors' => [...],    // Erreurs bloquantes
    'warnings' => [...]   // Avertissements non-bloquants
]
```

| Méthode | Route                                               | Description       |

## 🧪 Tests manuels à effectuer

1. **Export Excel Dossiers**

   - Se connecter en tant qu'archiviste
   - Aller sur `/archivist/import-export/`
   - Cliquer "Excel (.xlsx)" dans section Export > Dossiers
   - Vérifier le téléchargement et l'ouverture du fichier

2. **Export CSV Fichiers**

   - Cliquer "CSV" dans section Export > Fichiers
   - Ouvrir le fichier avec Excel/LibreOffice
   - Vérifier l'encodage UTF-8 et le séparateur `;`

3. **Télécharger Template Dossiers**

   - Cliquer "Télécharger le template d'import" dans section Import > Dossiers
   - Vérifier la structure : en-têtes (ligne 1), descriptions (ligne 2), exemple (ligne 3)

4. **Import Dossiers**

   - Remplir le template avec 3 dossiers de test
   - Uploader le fichier
   - Vérifier les messages de succès
   - Contrôler en base que les dossiers sont créés

5. **Import avec erreurs**

   - Créer un fichier avec ligne sans libellé
   - Créer une ligne avec département_id inexistant
   - Vérifier les messages d'erreur détaillés avec numéros de ligne

6. **Audit Logs**
   - Aller sur la page des logs d'audit (admin)
   - Vérifier qu'un log "Export Dossiers" apparaît avec le format et le nombre d'éléments
   - Vérifier qu'un log "Import Fichiers" apparaît avec les compteurs
